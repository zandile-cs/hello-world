# Student Management System (Java)

## 📌 Overview
This is a simple console based Student Management System built using Java.  
It demonstrates Object-Oriented Programming (OOP) principles and basic data management using ArrayLists.

---

## 🚀 Features
- Add a student
- Display all students
- Search student by ID

---

## 🛠 Technologies Used
- Java
- Object-Oriented Programming (OOP)
- ArrayList
- Scanner (for user input)

---

This project was built as part of my learning journey in Computer Science to improve my understanding of Java OOP concepts and version control using Git and GitHub.
